# Lista de comandos

| git init | inicializa la carpeta como un repositorio git |
| --- | --- |
| git add <file> | agrega al repo un archivo <file> o . para todo el directorio. |
| git commit -m “Mensaje” | Envía los cambios a la base de datos local de cambios (Staging). |
| git status | Para saber el estado de los cambios actuales. |
| git show | Todos los cambios hechos.  |
| git log <file> | Muestra el histórico de cambios de un archivo. |
| git push | Enviar cambios al servidor remoto. |
| git pull | Descargar cambios remotos. |
| git rm --cached <file> | Borra un archivo incluido en el stage con add <file> |
| git fetch | Actualiza los cambios remotos, es como un refresh |
| git cherry-pick <hash> -e | Me jalo el commit con el <hash> dado. |
| git checkout -t origin/feature/ejemplo | Checkout de una rama |
| git switch feature/ejemplo | Cambio a una rama descargada. |
| git clone <repo> | Clonar o descargar un repositorio <repo> |

<file> es el nombre de un archivo.
