| ' ' | Single quotes |

| `` | Backquotes, Backtick |
| . | point(numbers, versions), dot (domain names, object notation), period (finishing paragraph) |
| , | comma |
| : | colon |
| ; | semicolon |
| + | plus |
| - | minus |
| * | asterisk, times |
| / | division |
| - | hyphendash (normal, n-dash, m, dash; also for spaces) |
| _ | underscore |
| **~** | tilde |
| **!** | bang, exclamation sign |
| **@** | at sign |
| **#** | hash, octothorpe |
| **^** | caret |

| \ | backslash (useful for paths in Windows) |
