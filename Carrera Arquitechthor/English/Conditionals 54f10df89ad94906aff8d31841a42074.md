# **Zero Conditional**

- Obvious things
- Always happens. In this case, you can use **when**.
- If present simple, present simple.
- Real possibility.
- If we drive, we get there faster.

# First Conditional

- Real Possible Future.
- If present simple, future simple.
- imaginary, unreal, hypothetical.
- if we drive, we’ll get there faster.

# Second conditional

- If past simple, modal verb (would, could, might, should…) + infinitive.
- For all subjects is used were. I were, she were.
- if we drove, we’d get there faster.

# Third conditional

- If past perfect, would have + past participle.
- Impossible situation, impossible result. In the past.
- if we had driven, we would have gotten there faster.
