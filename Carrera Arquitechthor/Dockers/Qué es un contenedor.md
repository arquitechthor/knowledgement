# ¿Qué es un contenedor?

Analogía de contenedor.

**Containerización**: empleo de contenedores para construir y desplegar software.

**Flexible**. sirve para cualquier software.

**Livianos**. Se reutilizan cosas del sistema operativo.

**Potables**. Funcionan igual en cualquier máquina.

**Bajo acoplamiento**. Son autocontenidos. Tiene todo lo que necesita.

**Seguros**. El software no tiene acceso fuera de su contenedor.

**Escalables**. Permite ajustar sus parámetros sin afectar el software.

Diferencia entre contenedores y máquinas virtuales.
